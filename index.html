web liquid glass , by jvalaj :)

after seeing apple’s latest liquid glass ui update, i wanted to recreate a similar effect on the web using only html, css, svg, and javascript. i really love apple’s creativity and how much care goes into their interface design. this project builds liquid glass visuals through layered blur, refraction, subtle chromatic aberration, and interaction-driven motion, all without webgl or external libraries. focus is on performance, visual fidelity, and attention to detail using native web primitives.
